Multivariate Anomaly Detection for Memory Leak Identification

Problem

Memory leaks in long-running JVM applications degrade performance gradually and are difficult to detect early.

Data

JVM heap usage, garbage collection metrics, and system telemetry under simulated leak and non-leak conditions.

Approach

Trained LSTM models on normal behaviour and compared them with density-based approaches such as DBSCAN.

Results

LSTM models detected anomalies earlier and with fewer false positives.

What I Learned

Temporal modelling is essential for system-level anomaly detection problems.
